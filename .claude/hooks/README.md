# Hooks מקומיים - FlowPlan

תיקייה זו מכילה hooks מותאמים אישית עבור הפרויקט.

## 📝 document-session.js

**מה זה עושה:**
- רץ אוטומטית אחרי כל פעולה (PostToolUse hook)
- מתעד בזמן אמת כל פעולה: Edit/Write/Read/Bash וכו'
- כותב לוג בעברית ל-`.claude/session-log.md`

**פורמט הלוג:**
```markdown
# Session Log

## תאריך

- 14:30:45 ✏️ ערך: `src/services/auth.ts`
- 14:30:50 📝 יצר/כתב: `.claude/hooks/document-session.js`
- 14:31:00 🔧 הריץ: `npm run test`
```

**איך זה עובד:**
1. Hook Stop מופעל אחרי כל תגובה
2. הסקריפט מקבל את פרטי הכלי ששימש
3. מעצב רשומת לוג בעברית עם timestamp
4. מוסיף ל-`session-log.md`

**הגדרה:**
ראה `.claude/settings.json` → `hooks.Stop`

## 🎯 יתרונות

1. **מעקב אוטומטי** - לא צריך לזכור לתעד ידנית
2. **עברית** - קל לקרוא ולהבין
3. **כרונולוגי** - ממוין לפי תאריך ושעה
4. **לא מפריע** - רץ בשקט ברקע

## 📊 שימוש

פתח את `.claude/session-log.md` כדי לראות מה נעשה בסשן הנוכחי.

הקובץ מתעדכן אוטומטית עם כל פעולה!
